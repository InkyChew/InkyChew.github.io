Sorting algorithm - Merge sort

Merge sort

  1. 重複將資料從中間切成左右兩半，直到切出來的每一半只剩下一個元素
  2. 由小到大合併左右兩半
    • 新增左右兩半兩個陣列
    • 比較左右兩半元素的大小，進行合併

Time complexity

Worst case: O(n logn)
Best case: O(n logn)
divide() = log n ; merge() = n
