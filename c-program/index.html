C語言的那些事

指標 Poiter

指標存記憶體位址
透過指標變數提取記憶位址
宣告指標變數 型態 *名稱

Call by value

將變數x、y的值傳給變數a、b。進行swap()之後，變數a、b的值改變，但變數x與y的值不變。

Call by reference

& 表變數的位址
* 指標變數中，儲存位址所指向的值

將變數x、y的記憶體位址傳給指標變數a、b。也就是說，使用指標變數a、b，儲存變數x、y的記憶體位址。進行swap()之後，指標變數a、b儲存的記憶體位址互換，因此變數x與y的值改變。
