Design pattern 可依照目的(Purpose)分成三大類

1. Creational

  • Singleton
  • Factory Method
  • Abstract Factory
  • Builder
  • Prototype

2. Structural

  • Facade
  • Bridge
  • Decorator
  • Adapter
  • Composite
  • Flyweight

3. Behavioral

  • Mediator
  • Strategy
  • State
  • Command
  • Iterator
  • Memento
  • Visitor
  • Observer
  • template method
  • Chain of responsibility
  • Proxy
  • Interpreter
