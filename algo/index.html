演算法 (Algorithm)

  • 解決問題的流程
  • 滿足五條件
    1. 輸入 可有可無
    2. 輸出 至少要有一個
    3. 有限性 有限步驟內解決問題
    4. 明確性 步驟清楚明確、條理分明
    5. 有效性 必須能夠解決問題
  • 表示方式
    1. Psuedo code
    2. Flow chart
