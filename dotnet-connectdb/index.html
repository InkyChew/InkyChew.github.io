.Net 6 - Operate database by EFCore

Operate database by EFCore in .NET

The Entity Framework (ADO.NET Entity Framework) is a set of technologies in ADO.NET.
In the following article, I will introduce two methods, Database first and Code first, to Connect SQL Server DB in ASP.NET6 Web API by Entity Framework Core. Either one can achieve our goal. Let’s start!

DB First

  1. Add Model

    • package required

    • Open Package Manager Console

    • Use Scaffold-DbContext to generate code for a DbContext and entity types from database.

      This will create a class which extends DbContext

      Also, Entity type Record, which is a table in database.

  2. Add ConnectionStrings

    • package required

    • Add ConnectionStrings in appsettings.json
      Set the database’s server location and the database’s name.

    • Add db connection in Program.cs

  3. Add Controller

Code First

  1. Add Models Folder
    create DbContext and Model class

    • Record.cs
    • Accounting.cs

  2. ConnectionStrings

    • package required

    • Add ConnectionStrings in appsettings.json
      Set the database’s server location and the database’s name.

    • Add db connection in Program.cs

  3. Add Migration

    • package required
    • Add-Migration InitialCreate

      After the command is excuted, a new folder Migrations will be created.
    • Look into class InitialCreate in InitialCreate.cs There is an Up method to create table and a Down method to drop table.

  4. Execute Update-Database.

    • The command will create the database if it is not exsit.
    • Open the SQL Server management to see the created database.
