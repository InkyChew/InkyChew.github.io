結論

網頁中的日期與時間該如何處理？

將日期與時間以零時間偏移的UTC標準時間儲存，取得資料後依特定時區顯示。

日期與時間在 Angular 中的處理

使用 JavaScript 的 new Date() 方法產生 Date 型態的物件， 當 Date 型態物件轉換為 JSON 格式時，相當於轉換為以 ISO 8601 格式表示的 UTC 標準時間。

const today = new Date();
const jsonString = today.toJSON();
const isoString = today.toISOString();
console.log(jsonString === isoString); // true

Angular Date pipe 轉換時間

使用 Angular Date pipe 轉換 UTC 標準時間。語法如下。

{{ value_expression | date [ : format [ : timezone [ : locale ] ] ] }}

以 ISO 8601 表示日期與時間的字串 value_expression，必須包含 Z ，以表示要依特定時區顯示；否則字串將視為本地時間而不進行任何轉換。

iso = "2024-03-06T06:26:09.868"
isoz = "2024-03-06T06:26:09.868Z";

<p>{{ iso }} : {{ iso | date: "yyyy-MM-dd HH:mm:ss" }}</p>
<p>{{ isoz }} : {{ isoz | date: "yyyy-MM-dd HH:mm:ss" }}</p>

<!--
2024-03-06T06:26:09.868 : 2024-03-06 06:26:09
2024-03-06T06:26:09.868Z : 2024-03-06 14:26:09
-->

timezone 預設是電腦系統時區，也可自行指定。

<!-- example1 -->
<p>{{ isoz }} : {{ isoz | date: "yyyy-MM-dd HH:mm:ss" }}</p>
<!-- example2 -->
<p>{{ isoz }} : {{ isoz | date: "yyyy-MM-dd HH:mm:ss" : 'UTC +8' }}</p>

第一種情況: 依使用者所在時區顯示。 例如: 電腦系統時區在台灣，則顯示台灣時間；電腦系統時區在美國，則顯示美國時間。

第二種情況: 依指定的特定時區顯示。 例如：電腦系統時區在台灣、美國、任何國家，統一以台灣時間（UTC +8）顯示。

日期與時間在 .Net6 的處理

WebAPI 接收到 JSON 物件，轉換成 DateTime 型態物件並以 UTC 標準時間表示。 但從 C# 將 DateTime 型態物件轉換為 JSON 物件時，並不會自動加上時區，因此必須手動加上 Z ，表示零時間偏移。

writer.WriteStringValue(value);
// "2024-03-05T08:31:20.403"
writer.WriteStringValue(value.ToString("yyyy-MM-ddTHH:mm:ss.fffZ"));
// "2024-03-05T08:31:20.403Z"

在 controller 當中自訂 JSON Converter，轉換時間與日期的型態與格式。

public class JsonDateTimeConverter : JsonConverter<DateTime>
{
    public override DateTime Read(ref Utf8JsonReader reader, Type typeToConvert, JsonSerializerOptions options)
    {
        if(reader.TokenType == JsonTokenType.String && DateTime.TryParse(reader.GetString(), out var dateTime))
        {
            return dateTime;
        }
        return default;
    }

    public override void Write(Utf8JsonWriter writer, DateTime value, JsonSerializerOptions options)
    {
        writer.WriteStringValue(value.ToString("yyyy-MM-ddTHH:mm:ss.fffZ"));
    }
}

// Program.cs
builder.Services.AddControllers()
	.AddJsonOptions(options =>
	{
		options.JsonSerializerOptions.Converters.Add(new JsonDateTimeConverter());
	});
